## 개발의의
* 자신의 결과물들을 공개적으로 상시 피로할 플랫폼 확보
* 종합적인 백엔드 실력 향상


## 환경
* 클라우드 : AWS(t2.micro)
* OS : Ubuntu 24.04
* 리버스 프록시 서버 : Nginx
* 격리환경 : Docker
* ASGI서버 : Daphne
* 웹프레임워크 : Django
* 데이터베이스 : MySQL
